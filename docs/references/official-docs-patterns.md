# 公式ドキュメント 頻出パターン・語彙集

## 語彙

### lightweight
- **意味**: 軽量な（少ないリソースで動作する）
- **用例**: JavaScript (JS) is a **lightweight** interpreted programming language.
- **出典**: MDN JavaScript トップページ

### interpreted
- **意味**: インタプリタ型の（コードを1行ずつ実行する方式）
- **用例**: JavaScript is a lightweight **interpreted** (or just-in-time compiled) programming language.
- **出典**: MDN JavaScript トップページ

### just-in-time compiled
- **意味**: JITコンパイルされる（実行時にコンパイルする方式）
- **用例**: JavaScript is a lightweight interpreted (or **just-in-time compiled**) programming language.
- **出典**: MDN JavaScript トップページ

### first-class functions
- **意味**: 第一級関数（関数を変数に代入したり引数に渡したりできること）
- **用例**: ...a programming language with **first-class functions**.
- **出典**: MDN JavaScript トップページ

### well-known
- **意味**: よく知られた
- **用例**: It is most **well-known** as the scripting language for Web pages.
- **出典**: MDN JavaScript トップページ

### non-browser environments
- **意味**: ブラウザ以外の環境（"non-" は否定の接頭辞）
- **用例**: Many **non-browser environments** also use it.
- **出典**: MDN JavaScript トップページ

## 表現パターン

### X is a ... with ...
- **意味**: 「Xは〜を備えた...である」技術の定義パターン
- **用例**: JavaScript is a lightweight interpreted programming language **with** first-class functions.
- **出典**: MDN JavaScript トップページ

### While ..., ... also ...
- **意味**: 「〜として知られているが、〜でもある」対比表現
- **用例**: **While** it is most well-known as the scripting language, many non-browser environments **also** use it.
- **出典**: MDN JavaScript トップページ

### such as A, B and C
- **意味**: 「例えばA、B、Cなど」具体例の列挙
- **用例**: ...many non-browser environments also use it, **such as** Node.js, Apache CouchDB **and** Adobe Acrobat.
- **出典**: MDN JavaScript トップページ
